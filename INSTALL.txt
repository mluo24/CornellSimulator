To install: 

Make a switch with `opam switch create cs3110-2021fa ocaml-base-compiler.4.12.0`

Initialize the switch with the following libraries using this command: `opam install -y utop odoc ounit2 qcheck bisect_ppx menhir ocaml-lsp-server ocamlformat ocamlformat-rpc`

Then run:

`opam install Graphics`
`opam install js_of_ocaml`
`opam install js_of_ocaml-lwt`
`opam install lablgtk`
`opam install camlimages`
`opam install ANSITerminal`

make build 
make play

Check the README.md to see how to play the game.
